## Стиль
Ощущение: тёмный, премиальный, ручная работа
Фон: тёмный (#1c1006)
Акцентный цвет: #c4a060 (золото)
Шрифт заголовков: Poiret One (Google Fonts)

## Технический стек
- Один HTML-файл (index.html), CSS и JS внутри
- Desktop-first, breakpoints: max-width 1024px / 640px
- Фото: /photos-optimized/ (WebP + JPG), все обёрнуты в <picture>
- Юридические документы: privacy.html, consent.html (из папки Soglasiya/)

## Форма
- Cloudflare Worker: задеплоить отдельно, URL вставить в JS (переменная workerUrl)
- Поля: Имя, Телефон (обязательные), Email (необязательный), Формат (select), Объём/пожелания (textarea)
- Два обязательных чекбокса: политика конфиденциальности + согласие на обработку ПД

1. Hero — слайдер 4 фото, анимированный заголовок, CTA

3. Три цитруса — флагманский продукт
4. Галерея — 10 фото в сетке
5. Для кого — 5 сегментов (сегменты b2b)
6. Кейсы — 3 реальных кейса
7. Ассортимент — 4 категории
8. Почему мы — 4 преимущества с цифрами
9. Основатель — фото + текст
10. Форма — заявка + контакты
11. Футер

## Решения, которые уже приняты
- Hero: тёмный слайдер с 4 фото, не статичное изображение
- CTA: только Telegram (@KUTKEVICH), без прямой ссылки на покупку
- Галерея: CSS Grid с фиксированными строками, не карусель
- Кастомный cursor-ring (только desktop, hover:hover)
- Typewriter-анимация на цитате в секции цитрусов
- Scroll-reveal через IntersectionObserver, обёрнут в prefers-reduced-motion
- WebP + JPG fallback через <picture> для всех фото
- Skip-link и :focus-visible добавлены (доступность)
- privacy.html и consent.html — полные юридические документы по 152-ФЗ
